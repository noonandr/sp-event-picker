# Seat Plan Event Picker
* Responsive UI component to select event type, date and time *

## Installation

cd into root folder:

```
$ cd sp-event-picker
```

Install npm modules:


```
$ npm install
```

## Run Server

To run the server:

```
$ npm run serve
```
